# YummyMummy -A recipe web app
# Technologies

* Asp.Net Core
* IIS
* MVC
* SQL Server Express Database
* Entity Framework
* Bootstrap css
* Azure

# Features
* User Registration
* User Login
* Category CRUD
* Ingredient CRUD
* Recipe CRUD
* Recipe Review CRUD
* Menu CRUD
* Search by recipe name
* Pagination
